### 日志模块开发 20240111-20240115

日志模块
```
1. 日志级别: DEBUG INFO ERR
2. 日志输出文件名（支持日期命名），日志滚动（若日志文件过大，是输出到新文件？还是覆盖？）
3. c 格式化风格 printf + 符号，比c++的流式输出更加便利
4. 线程安全(待完善)
```

LogLevel: 
使用枚举类型来定义日志级别，共三个级别
```
DEBUG
INFO
ERROR
```

LogEvent:
```
文件名、行号
MsgNo: rpc请求信息的符号(待完善)
线程id、进程号
日期与时间 精确到毫秒
自定义消息: 输出的字符串信息
```

```
日志格式
[LogLevel][%y-%m-%d %H:%M:%S]\t[pid:thread_id]\t[filename:line][%msg]
```
